我的工具人 1 - Notion 集成
一键同步抖音小视频数据与游戏打金收益，高效管理内容创作与副业收入
✨ 核心功能
🎬 抖音小视频管理
数据自动同步：自动抓取抖音账号视频数据（播放量、点赞、评论、转发），同步到 Notion 数据库
内容排期：在 Notion 中规划发布日历，支持选题、脚本、素材、发布状态全流程管理
数据分析看板：自动生成视频表现报表，直观对比不同内容的流量效果，辅助选题决策
达人协作：支持多人协同编辑，适合 MCN 机构或团队化运营
🎮 游戏打金收益追踪
多游戏支持：覆盖《三角洲行动》等主流打金游戏，记录角色、服务器、产出物品
收益自动核算：自动计算金币 / 装备兑换比例，统计每日 / 每周 / 每月净收益（扣除点卡、脚本成本）
时间投入分析：追踪每小时产出效率，优化打金路线与时间分配
库存管理：在 Notion 中管理游戏物品库存，支持出售 / 囤积状态标记
🚀 快速开始
安装集成：在 Notion 应用市场中搜索「我的工具人 1」，点击「使用」
授权连接：按提示授权你的抖音开放平台账号与游戏角色信息
选择模板：一键导入预设的「抖音内容管理」与「游戏打金收益」数据库模板
开始同步：设置同步频率（实时 / 每小时 / 每日），数据将自动更新到你的 Notion 页面
